# 📘 TABLA DE REGLAS DE INTEGRALES 

## 🔹 1. Reglas Básicas (Polinomios y Linealidad)

| Tipo                  | Regla                                   | Ejemplo           | Resultado                        |
| --------------------- | --------------------------------------- | ----------------- | -------------------------------- |
| Potencia              | $\int x^n dx = \frac{x^{n+1}}{n+1} + C$ | $\int x^3 dx$     | $\frac{x^4}{4}+C$                |
| Constante por función | $\int a f(x) dx = a \int f(x) dx$       | $\int 3x^2 dx$    | $x^3+C$                          |
| Suma/Resta            | $\int (f+g)dx = \int f dx + \int g dx$  | $\int (x^2-3x)dx$ | $\frac{x^3}{3}-\frac{3x^2}{2}+C$ |

✔ Se aplica directamente en integrales como
$\int_0^2 (3x^2 - 2x) dx$

## 🔹 2. Exponenciales

| Tipo               | Regla                                  | Ejemplo          | Resultado             |
| ------------------ | -------------------------------------- | ---------------- | --------------------- |
| Exponencial básica | $\int e^x dx = e^x + C$                | $\int e^x dx$    | $e^x+C$               |
| Con constante      | $\int e^{ax} dx = \frac{1}{a}e^{ax}+C$ | $\int e^{2x} dx$ | $\frac{1}{2}e^{2x}+C$ |

## 🔹 3. Sustitución (Regla de la cadena invertida)

### 📌 Patrón clave

Si ves:

$f'(x) e^{f(x)} \quad \text{o} \quad f'(x)\cos(f(x))$

Entonces:

$\int f'(x) e^{f(x)} dx = e^{f(x)} + C$

### Procedimiento formal

1. $u = f(x)$
2. $du = f'(x)dx$
3. Sustituir

| Ejemplo               | Sustitución | Resultado                |
| --------------------- | ----------- | ------------------------ |
| $\int x e^{x^2} dx$   | $u=x^2$     | $\frac{1}{2}e^{x^2}+C$   |
| $\int x \cos(x^2) dx$ | $u=x^2$     | $\frac{1}{2}\sin(x^2)+C$ |

✔ Este tipo aparece varias veces en tus exámenes.

---

## 🔹 4. Integración por Partes

### 📌 Regla


$\int u,dv = uv - \int v,du$

### Elección práctica (LIATE)

Orden de prioridad para elegir (u):

1. Log
2. Inversa trig
3. Algebraica  ← aquí cae (x)
4. Trig
5. Exponencial

| Ejemplo           | Elección | Resultado               |
| ----------------- | -------- | ----------------------- |
| $\int x e^x dx$   | $u=x$    | $xe^x - e^x + C$        |
| $\int x\sin x dx$ | $u=x$    | $-x\cos x + \sin x + C$ |

✔ Muy importante para $x e^x$.

---

## 🔹 5. Trigonométricas con factor

| Regla             | Fórmula                  |
| ----------------- | ------------------------ |
| $\int \sin(ax)dx$ | $-\frac{1}{a}\cos(ax)+C$ |
| $\int \cos(ax)dx$ | $\frac{1}{a}\sin(ax)+C$  |

Ejemplo:

$\int \sin(2x) dx = -\frac{1}{2}\cos(2x)+C$


---

## 🔹 6. Fracciones Racionales

### 📌 Método General

1. Si grado numerador ≥ grado denominador → división
2. Factorizar denominador
3. Fracciones parciales:

   * $\frac{A}{x-a}$
   * $\frac{Ax+B}{x^2+1}$

### Fórmulas útiles

| Tipo                    | Integral |     |      |
| ----------------------- | -------- | --- | ---- |
| $\int \frac{1}{x} dx$   | $\ln     | x   | + C$ |
| $\int \frac{1}{x-a} dx$ | $\ln     | x-a | + C$ |

Ejemplo simple:

$\int \frac{1}{x(x-1)}dx$


Se descompone en:

$\frac{A}{x}+\frac{B}{x-1}$


✔ Este es el nivel más técnico.
